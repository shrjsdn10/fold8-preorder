# 폰스팟 Fold8 사전예약 랜딩 페이지

`index.html`을 브라우저에서 열면 실행되는 독립 보관본입니다.

## 구성

- `index.html`: 랜딩 페이지 전체 코드
- `assets/`: 히어로 및 제품 이미지 4개

## 현재 외부 링크

- 카카오 채널: https://pf.kakao.com/_hZPcG/friend
- 인스타그램: https://www.instagram.com/phonespot.kr/

## 남은 작업

- 푸터의 임시 전화번호 `000-0000-0000` 교체
- 공식 제품 이미지 공개 후 예상 렌더 이미지 교체
- 폴드8 이미지는 현재 가장자리 페이드 및 워터마크 영역 보정 방식 사용
